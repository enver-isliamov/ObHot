## Как запустить установку на новом сервере
Теперь вам не нужно ничего делать на сервере вручную. После покупки нового VPS просто введите одну команду:

## Bash
curl -sL https://raw.githubusercontent.com/enver-isliamov/my-vpn-stack/main/setup.sh | bash

## Почему это *«идеальное»* решение:
Чистота кода: В самом скрипте больше нет бесконечных строк HTML/CSS. Он просто копирует их из вашей папки website [cite: 28].

Легкость правок: Если вы захотите изменить дизайн сайта-заглушки, вы просто меняете файлы в папке website на GitHub. При следующей установке на новом сервере скрипт сам скачает новую версию [cite: 60].

Полная автоматизация: Скрипт сам генерирует уникальные ключи Reality и UUID для каждого нового сервера, что делает их максимально защищенными [cite: 143].

Скорость: Развертывание всего стека (Nginx + VPN + AdGuard + MTProto) занимает меньше 3-х минут [cite: 145].
